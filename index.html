Visualizing Mumbai Open Data

What can we find out about Mumbai from publicly available data? How does the city compare with other cities across various parameters? Can open-data help uncover new insights about the ever-growing nature of the city? How can we create stories powered by this data that a common man can relate to? This project aims to tackle these and several other questions.
Principal Investigator: Prof. Venkatesh Rajamanickam
